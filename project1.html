Telco Customer Churn Prediction

Developed an interactive dashboard and predictive model to identify customers likely to churn. Achieved 77% accuracy using a Random Forest model.

Problem Statement

Customer churn is a critical issue in the telecommunications industry. This project aims to identify the key factors influencing customer churn and develop a predictive model to help retain at-risk customers.

Methodology

  • Cleaned and preprocessed the dataset, addressing missing and inconsistent values.
  • Conducted exploratory data analysis to identify trends, such as the impact of tenure and monthly charges on churn.
  • Used SMOTE to handle imbalanced data for better model performance.
  • Built a Random Forest model and fine-tuned hyperparameters using GridSearchCV.

Key Insights and Results

  • Achieved 77% accuracy in predicting customer churn.
  • Key contributors to churn include rude support, contract type, monthly charges, and tenure.
  • Customers with with Fiber Optic Internet Service and no technical support are more likely to churn.
  • Customers with short tenures and high monthly charges are more likely to churn also.

Technologies Used

Python, pandas, scikit-learn, Matplotlib, Seaborn, SMOTE, Random Forest, GridSearchCV.
