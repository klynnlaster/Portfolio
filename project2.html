Interactive YouTube Quality Content Analysis Tool

An application for sentiment analysis on YouTube comments, achieving 94% accuracy in predicting whether a comment is positive or negative.

Problem Statement

Analyzing user feedback is critical for understanding audience engagement and content quality. This project addresses the need for automating sentiment analysis to classify comments and uncover trends in user sentiment.

Methodology

  • Data Cleaning and Preprocessing: Removed special characters, links, and missing values.
  • Feature Engineering: Applied TF-IDF vectorization to extract features from text data.
  • Sentiment Analysis: Used the VADER library for initial analysis and logistic regression for predictive modeling.
  • Model Evaluation: Achieved 94% accuracy using precision, recall, and F1-score metrics.

Key Insights and Results

  • Model achieved 94% accuracy in predicting comment sentiment.
  • Positive comments often included terms like "love," "amazing," and "great."
  • Negative comments highlighted issues with terms like "hate," "dislike," and "bad."

Challenges and Learnings

Managing missing or noisy data was challenging. Cleaning text and handling special characters required robust preprocessing techniques. Another challenge was ensuring model interpretability, which was addressed by visualizing the most frequent words in positive and negative comments.

Technologies Used

Python, pandas, scikit-learn, Matplotlib, Streamlit, VADER Sentiment Analyzer.
